Overview Table of CR models using Copulas

Introduction

This table has been created as part of the manuscript: Competing risk models using copulas: A comprehensive review with the aim of consolidating articles that has covered CR models using copulas. The provision of open-source and well-documented software is essential to ensure the widespread adoption of new statistical methods. Reproducibility is crucial in the development and evaluation of new methods. The table provides such details, and was constructed using R package: reactable (Lin, 2023).

Explanation of the table columns

Explanation of table column names:

  • title title of the paper
  • truncation type of truncation
  • censoring type of censoring
  • model type of marginal survival functions used; ML - Machine learning, non - non-parametric, par - parametric model.
  • field Field of study survival, meta analysis or reliability
  • type copula type, including Arch - Archimidean, H Arch - hierarchical Acrhimidean, Gaus - Gaussian, FGM - Farlie–Gumbel–Morgenstern, Frank, AMH - Ali-Mikhail-Haq.
  • covariates whether covariates are used or not
  • code availability of code used to generate results
  • authors name of author(s) (hidden)
  • year year the article was published (hidden)
  • paper link to the paper (hidden)
  • dataset Whether real dataset(s) were utilised (hidden)
  • simulation Whether simulation study was utilised (hidden)
  • CR type type of CR: CR - for latent CR type; CR-observ - Observable quantities type of CR; semi-CR - latent type CR in which a non-terminal event ca be censored by a terminal event, and Cens - dependent censoring type of analysis.
